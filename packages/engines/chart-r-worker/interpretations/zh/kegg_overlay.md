# KEGG 通路图叠加（KEGG Pathway Map Overlay）

## 图表说明

KEGG 通路图叠加在官方 KEGG 代谢通路图的框架上，将差异代谢物节点用颜色标注：上调代谢物节点变为红色/橙色，下调代谢物变为蓝色/绿色，未检测到或无差异代谢物保持灰色。可直观呈现受扰代谢通路的全景，显示哪些反应节点被激活或抑制。

## 如何解读

- **红/橙色节点**：在该通路中对应上调的代谢物或反应，提示该步骤代谢流增强
- **蓝/绿色节点**：对应下调的代谢物，提示代谢流减弱或通路受抑制
- **连续的颜色块**：若通路中相邻节点均变色，说明该代谢通路多个步骤同时受到调控
- **孤立变色节点**：仅单个节点变色，可能是该代谢物参与多条通路，需结合其他通路一起解读
- **未着色（灰色）区域**：未检测到差异或未注释，不能推断该代谢物无变化

## 注意事项

- KEGG 通路图叠加需要准确的 KEGG Compound ID，注释质量直接影响结果
- 通路图是静态的 KEGG 框架，不反映真实的代谢通量（需要同位素示踪等方法）
- 同一代谢物可能出现在多个通路中，需避免过度解读单一通路图
- 该图依赖 pathview 或 ggkegg 包，需要网络访问 KEGG API

# PLS-DA 得分图（PLS-DA Score Plot）

## 图表说明

偏最小二乘判别分析（PLS-DA）得分图是有监督的样本分群可视化，利用分组信息最大化组间分离。图中展示前两个潜变量（LV1/LV2）下的样本位置，并标注模型质量指标：R²X（解释的 X 方差）、R²Y（解释的 Y 方差）和 Q²（交叉验证预测能力）。

## 如何解读

- **样本分离**：组间分离越清晰，说明代谢谱差异越显著；PLS-DA 比 PCA 通常分离效果更好
- **R²X**：模型使用的特征信息量，越高说明提取的样本信息越全面
- **R²Y**：模型对分组的解释程度，越高说明代谢特征与分组关联越强
- **Q²**：交叉验证后的预测能力，Q² > 0.5 说明模型有良好预测能力，Q² < 0 提示过拟合
- **椭圆**：各组的 95% 置信区间，不重叠说明组间代谢谱有显著区分

## 注意事项

- PLS-DA 是监督方法，即使数据为随机噪声也能产生表观分离，必须通过置换检验（permutation test）验证模型有效性
- Q² 应显著高于置换检验得到的 Q² 分布，否则模型无效
- R²Y >> Q² 时（差距 >0.3），提示过拟合，样本量可能不足
- 该图之后通常配合 VIP 棒棒糖图展示驱动分离的关键代谢物

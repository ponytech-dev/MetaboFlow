# 缺失值分布图（Missing Value Heatmap）

## 图表说明

缺失值分布图以热图形式展示特征×样本矩阵中 NA 值（缺失检测）的分布模式，黑色或深色表示缺失，白色或浅色表示检测到值。通过可视化缺失的系统性模式，帮助判断缺失的原因（随机缺失 vs 结构性缺失）和选择合适的填充策略。

## 如何解读

- **随机分散的缺失（盐椒噪声）**：缺失值随机分布，提示低丰度特征未被一致检测，可用 KNN 或最小值填充
- **整列缺失（某样本全缺失）**：该样本可能进样失败或数据损坏，需排查后决定是否纳入分析
- **整行缺失（某特征全缺失）**：该特征在所有样本中均未检测到，应在过滤步骤中去除
- **组特异性缺失**：某特征在一组中全部缺失但另一组中全部检测到，可能是该代谢物浓度低于某组检测限，具有生物学意义
- **缺失比例**：通常要求特征在 ≥70% 样本中有检测值，否则过滤

## 注意事项

- 缺失不等于浓度为零，代谢组学中"未检测到"通常意味着低于检测限
- 组特异性缺失特征在差异分析时需特殊处理，不能简单填充为 0 或均值
- 样本缺失率 >30% 通常建议排除该样本
- 缺失填充方法（最小值、KNN、RF 等）的选择会影响后续统计结果

# 时序模糊聚类图（Mfuzz Temporal Clustering Plot）

## 图表说明

Mfuzz 时序聚类图用于时间序列代谢组学实验，将代谢物按时序表达模式（如 0h→6h→24h→72h）聚为 N 个模糊聚类（soft clustering），每个聚类展示一个子图：灰色细线为单个代谢物轨迹，彩色粗线为聚类平均趋势（±SD），颜色深浅反映该代谢物对该聚类的隶属度分值。

## 如何解读

- **平均趋势线（彩色粗线）**：代表该聚类内代谢物的典型时序变化模式（上升、下降、先升后降等）
- **隶属度颜色**：深色轨迹表示该代谢物对该聚类的隶属度高（高确定性），浅色表示模糊隶属（可能属于多个聚类）
- **聚类数 N**：需根据数据特点预先设定，建议结合轮廓系数等指标选择最优 N
- **横轴（时间点）**：对应实验设计中的时间序列，应标注具体时间单位
- **纵轴（标准化丰度）**：通常为 Z-score，不代表绝对浓度

## 注意事项

- 模糊聚类允许一个代谢物同时属于多个聚类，不同于硬聚类方法（k-means）
- 时间点 <3 个时，Mfuzz 聚类意义有限，建议使用至少 4 个时间点
- 每个聚类的代谢物列表可进行通路富集，揭示哪条通路在什么时间点被激活
- 初始聚类中心会影响最终结果，应设置随机种子保证可重复性

# 多面板质控仪表板（Multi-panel QC Dashboard）

## 图表说明

质控仪表板将多个关键质控图表组合为 2×2 或 3×2 的多面板布局：通常包含①样本 QC 强度箱线图、②QC RSD 分布直方图、③PCA 样本得分图（QC 高亮）、④缺失值比例条形图或缺失值热图。单图全面评估数据集的整体质量，适合作为数据分析报告的首页。

## 如何解读

- **面板①（强度箱线图）**：评估进样量一致性和批次效应，QC 样本的中位强度应稳定一致
- **面板②（RSD 分布）**：评估特征重现性，80% 以上的特征 RSD<20% 为达标
- **面板③（PCA 图）**：评估样本分群质量，QC 样本应聚集于中心，各实验组分开
- **面板④（缺失值图）**：评估数据完整性，样本级和特征级缺失率应均在可接受范围
- **综合判断**：四个面板综合通过，数据质量符合后续分析要求；任一面板异常需重点调查

## 注意事项

- 仪表板适合快速整体评估，发现问题后应查看各子图的详细版本
- 不同分析批次或实验的 QC Dashboard 对比，能快速发现系统性差异
- 面板组合可根据实验设计调整，如有 TIC 数据可替换其中一个面板
- 仪表板中每个面板应保持简洁，不需要过多细节，以全局视角为主

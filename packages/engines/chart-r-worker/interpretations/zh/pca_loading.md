# PCA 双标图（PCA Biplot / Loading Plot）

## 图表说明

PCA 双标图在样本得分图的基础上叠加特征 loading 向量，同时展示样本分布和驱动该分布的关键特征方向。每个箭头代表一个特征，箭头方向指向该特征使样本得分升高的方向，箭头长度反映该特征对主成分的贡献程度。

## 如何解读

- **样本点**：与 PCA 得分图含义相同，反映样本间代谢组相似性
- **特征箭头（loading 向量）**：箭头越长说明该特征在 PCA 分离中贡献越大
- **方向一致性**：与某组样本同方向的长箭头所代表的特征，在该组中丰度偏高
- **相互垂直的箭头**：代表两个特征在 PCA 空间中不相关，可能属于不同代谢通路
- **相反方向的箭头**：代表在某一组中升高但另一组降低的特征，即典型的反相关代谢物

## 注意事项

- 特征数量多时，双标图会显得拥挤，通常只标注 Top 20–50 个贡献最大的特征
- loading 向量的长度受数据归一化方式影响，不同归一化下比较应谨慎
- 双标图适合探索性分析，定量差异应通过差异分析（火山图）确认
- 若需要更精确的特征重要性排序，建议使用 PLS-DA + VIP 分析

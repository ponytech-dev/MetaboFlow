# 样本相关性热图（Sample Correlation Heatmap）

## 图表说明

样本相关性热图展示所有样本两两之间的 Pearson 或 Spearman 相关系数矩阵，颜色从深蓝（低相关或负相关）到深红（高正相关）渐变。行列均按层次聚类排序，帮助直观识别样本分组、批次聚类和异常样本。

## 如何解读

- **对角线**：始终为红色（相关系数=1，每个样本与自身完全相关）
- **同组样本块**：同一实验组的样本应形成深红色方块，说明组内代谢谱一致
- **组间相关性**：不同组之间的方块颜色越浅，说明组间代谢谱差异越大
- **异常样本**：若某样本与同组其他样本的相关性低（浅色行/列），应标注为潜在离群值
- **批次聚类**：若不同实验组但同批次的样本聚集在一起，提示批次效应显著

## 注意事项

- 相关系数 <0.9 的样本对（同实验组内）通常值得关注，<0.8 则需要重点排查
- 该图对数据归一化敏感，未归一化的数据可能因稀释度差异导致假相关
- 大规模研究（>100 个样本）中，热图会变得拥挤，可用散点矩阵或亚组展示替代
- 批次校正效果可通过对比校正前后相关性热图的聚类模式来评估

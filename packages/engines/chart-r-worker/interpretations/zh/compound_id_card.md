# 化合物身份证（Chemical Identity Card）

## 图表说明

化合物身份证是单化合物注释证据的多面板综合展示，通常包含 4–6 个子图：①分子结构（SMILES 渲染）、②MS2 镜像比对图、③同位素模式验证图、④EIC 提取离子色谱图、⑤关键注释指标汇总表（置信等级、精确质量误差、余弦相似度、RT 偏差）。所有证据汇聚于一图，提供最全面的注释验证视图。

## 如何解读

- **分子结构图**：展示候选化合物的 2D 结构，直观呈现官能团和骨架
- **MS2 镜像图**：验证碎裂模式，余弦相似度应 >0.7
- **同位素验证图**：确认分子式，M+1/M+2 比值应与理论一致
- **EIC 图**：确认在目标 RT 处有清晰、对称的色谱峰
- **汇总指标**：Schymanski 等级、精确质量误差（ppm）、RT 偏差（min）、MS2 相似度分数一目了然
- **综合判断**：所有证据均满足时给予高置信注释，任一证据不足时降低等级或标注"tentative"

## 注意事项

- 化合物身份证是论文补充材料中最有说服力的注释支撑材料
- 需要原始 mzML 数据（EIC）、MS2 谱图数据和 RDKit/rcdk 进行结构渲染
- 一张身份证仅适用于一个候选化合物，不用于并排比较多个候选
- 若无法渲染分子结构，可用 InChI Key 或 SMILES 字符串代替

# 综合分析仪表板（Comprehensive Analysis Dashboard）

## 图表说明

综合分析仪表板是代谢组学论文主图的标准四联组合格式：左上为 PCA 得分图（无监督样本分群）、右上为火山图（差异代谢物全局分布）、左下为 Top 差异代谢物热图（表达模式聚类）、右下为通路富集气泡图（功能注释）。四个面板覆盖从样本质量→差异发现→代谢模式→功能解读的完整分析故事。

## 如何解读

- **左上（PCA）**：验证样本分群是否与实验设计一致，组内变异是否可接受，是否存在异常样本
- **右上（火山图）**：识别显著上调和下调代谢物的数量、分布和已注释名称，快速了解差异规模
- **左下（热图）**：展示 Top 差异代谢物的跨样本表达模式，确认差异的组内一致性和组间差异
- **右下（通路气泡图）**：将差异代谢物映射到功能通路，回答"这些代谢变化影响了哪些生物学过程"
- **四图联读**：从 PCA 确认质量 → 火山图发现差异 → 热图验证模式 → 通路图解读功能，构成完整的科学叙事

## 注意事项

- 四联仪表板适合单对比（一个实验组 vs 一个对照组），多对比应分别展示
- 各面板大小比例应均衡，建议 2×2 等分或根据信息量调整
- 出版级图表需确保字体 ≥8pt、分辨率 ≥300 dpi、图例完整
- 该图通常作为论文 Figure 1 或 Figure 2，需要简洁精准，不宜信息过载

# 特征云图（m/z-RT Scatter Plot）

## 图表说明

特征云图在二维平面上展示全局检测到的所有特征，x 轴为保留时间（RT，分钟），y 轴为质荷比（m/z），每个点代表一个检测特征，点的颜色或大小编码该特征的最大强度或平均强度。该图提供整个代谢组数据集的鸟瞰视角。

## 如何解读

- **特征密度分布**：密集区域（通常在 RT 2–15 min）对应色谱分离效果好的保留时间段
- **点的大小/颜色**：越大越亮的点代表丰度越高的特征，通常是主要代谢物
- **稀疏区域**：高 m/z 区域（>1000 Da）特征较少，m/z < 100 区域多为溶剂离子
- **特征总数**：图中点的总数反映本次分析检测到的特征规模，正离子模式通常 2000–10000 个特征

## 注意事项

- 特征云图不区分真实代谢物与噪声特征，需经过峰过滤后才有生物学意义
- 聚集成簇的特征（相同 RT、不同 m/z）可能是同一化合物的不同加合物或同位素
- m/z 500–900 Da 之间的密集特征簇常见于磷脂类代谢物
- 不同批次的特征云图对比可快速发现 RT 漂移或质量轴偏移问题

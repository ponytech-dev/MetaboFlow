# 桑基图（Sankey / Alluvial Diagram）

## 图表说明

注释质量桑基图展示代谢组学分析流程中特征从检测到注释各阶段的数量流向：从原始检测特征 → MS1 质量过滤 → 数据库搜索命中 → Schymanski 等级分类。每个流向宽度与通过该节点的特征数量成正比，直观展示注释流程的各阶段"漏斗"效率和最终各置信等级的注释分布。

## 如何解读

- **节点（宽矩形）**：每个处理阶段，宽度代表在该阶段的特征总数
- **流（连接带）**：从一个节点流向下一个节点的特征，带宽代表数量
- **分叉流向**：在注释等级节点处，流向多个等级（Level 1/2/3 等）的宽度反映各等级的注释数量占比
- **漏斗收窄**：从左到右特征数量递减是正常的；收窄越快说明该步骤过滤/命中率越低
- **颜色编码**：不同颜色可区分不同化学类别或注释来源（如 HMDB、MassBank 等）

## 注意事项

- 桑基图是流程质量的汇总图，不展示单个代谢物信息
- "大量特征未命中数据库"时，需检查数据库覆盖范围和质量容差设置是否合理
- 该图常用于方法学论文的 Supplementary 部分，展示注释工作流的系统性
- 各等级注释比例可作为不同方法之间对比的质量指标

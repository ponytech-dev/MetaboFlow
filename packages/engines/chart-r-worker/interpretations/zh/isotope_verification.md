# 同位素模式验证图（Isotope Pattern Verification Plot）

## 图表说明

同位素模式验证图通过成对柱状图对比每个候选化合物的实验同位素分布（M, M+1, M+2, M+3）与理论计算值（基于分子式）。每个同位素峰对应一对柱：实验值（蓝色）和理论值（橙色），同时标注质量精度（ppm 误差）和同位素分布匹配分值。

## 如何解读

- **实验 vs 理论柱高比例**：两对柱的高度比例越接近，说明同位素分布越符合理论分子式
- **M+1/M 比值**：反映碳原子数量（每个 C 贡献约 1.1% M+1 强度），可快速估算分子中的碳数
- **M+2/M 比值**：含 S、Cl、Br 等杂原子时 M+2 峰会异常增高，可作为杂原子检测的快速筛查
- **ppm 误差标注**：M+0 峰的精确质量误差，<5 ppm 为合格，<2 ppm 为高精度质谱仪标准
- **同位素分值**：综合多个同位素峰的匹配程度计算，>0.8 表示强烈支持该分子式

## 注意事项

- 低丰度特征的 M+1、M+2 峰信噪比低，同位素分布不可靠，验证意义有限
- M+2 异常升高不一定是卤素，也可能是同 RT 的杂质峰叠加
- 高分辨质谱（HRMS）的同位素验证意义更大；低分辨质谱不建议强调该证据
- 该图是化合物身份证（A13）的重要组成部分

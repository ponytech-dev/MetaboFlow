# 提取离子色谱图（EIC）

## 图表说明

提取离子色谱图（EIC）在指定 m/z 窗口（目标质量 ± ppm 容差）内提取特定离子的强度随保留时间的变化，用于验证特定化合物的检测情况。图中标注检测到的峰及其积分区间，可叠加多个样本进行对比。

## 如何解读

- **单一清晰峰**：在预期保留时间处出现单一对称峰，表明该化合物被可靠检测
- **峰面积**：与化合物浓度成正比，组间峰面积差异反映定量差异
- **多峰出现**：同一 m/z 在不同 RT 处出现多个峰，可能是同分异构体或加合物
- **标注的 RT 和 m/z**：验证检测到的峰是否与理论值吻合（RT 偏差 <0.05 min，质量误差 <5 ppm）

## 注意事项

- EIC 的 ppm 窗口影响结果：窗口过窄可能漏检，过宽会引入干扰峰
- 若在目标 RT 处无峰，可能是该化合物浓度低于检测限，或发生了 RT 漂移
- 应同时查看正负离子模式的 EIC 以全面评估检测情况
- EIC 图是 MS2 注释验证的重要辅助证据，应与镜像谱图对比使用

# 分子网络图（Molecular Network）

## 图表说明

分子网络图基于 MS2 碎片谱图的余弦相似性构建节点-边网络：每个节点代表一个检测特征，两个节点之间的边代表其 MS2 谱图相似性超过阈值（通常余弦 >0.7）。结构相似的化合物自然聚集成团，节点颜色可编码化学分类、log2FC 或注释置信度，用于发现未注释代谢物的结构线索。

## 如何解读

- **聚类团（cluster）**：节点紧密连接的区域，代表具有相似碎裂模式的化合物族（通常为同一化学类别）
- **边的粗细/颜色**：反映两节点间的余弦相似度，越粗/越深代表越相似
- **孤立节点**：无边连接的节点，说明该特征的 MS2 与其他任何特征均不相似，可能是独特结构或 MS2 质量低
- **节点颜色（log2FC）**：红色节点在实验组中上调，蓝色下调，灰色无显著差异
- **已注释节点（标签）**：有化合物名称标注的节点，可作为所在聚类团的结构参考节点

## 注意事项

- 分子网络需要足够质量的 MS2 数据，MS2 覆盖率低时网络稀疏，解读价值有限
- 余弦相似度阈值影响网络密度：阈值过低导致过度连接，过高导致网络过于稀疏
- 聚类内未注释节点可根据已注释节点的结构推断可能的化学类别（结构相似原则）
- 大型网络建议进行模块化分析（如 Louvain 算法），避免视觉过度复杂

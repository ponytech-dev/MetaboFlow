# 森林图（Forest Plot）

## 图表说明

代谢组学森林图以荟萃分析风格展示多个代谢物的效应量（log2FC）和 95% 置信区间，每个代谢物对应一行，中心点为 log2FC 估计值，水平线段为 CI 范围，垂直参考线（log2FC=0）表示无效应。代谢物通常按化学超类分组排列，颜色区分不同超类。

## 如何解读

- **中心点位置**：log2FC 的点估计值，正值（参考线右侧）为上调，负值（左侧）为下调
- **水平线段（CI）**：置信区间不跨越 0 线的代谢物（完全在一侧）具有统计显著性
- **点大小（可选）**：有时用点的大小编码 -log10(adj.p)，越大越显著
- **颜色分组**：不同化学超类用不同颜色，方便识别特定化学类别的整体趋势
- **整体效应（菱形）**：若计算了整体效应量，菱形显示在所有代谢物行的最下方

## 注意事项

- 代谢组学中的"森林图"是借鉴荟萃分析形式，并非真正的荟萃分析，不涉及跨研究合并
- CI 宽度反映样本量和个体变异，CI 极宽的代谢物统计效力有限
- 建议同时标注 adj.p 值（如 FDR 校正）以便读者快速判断显著性
- 按化学类别分组可揭示"某类代谢物整体向同一方向变化"的模式

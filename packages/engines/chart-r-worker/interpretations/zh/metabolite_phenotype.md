# 代谢物-表型相关热图（Metabolite-Phenotype Correlation Heatmap）

## 图表说明

代谢物-表型相关热图展示选定代谢物（行）与临床/生理指标（列，如体重、血压、炎症因子、基因表达等）之间的 Spearman 或 Pearson 相关系数。颜色从深蓝（强负相关）到深红（强正相关），每个格子标注 r 值，显著关联加星号（*p<0.05，**p<0.01，***p<0.001）。

## 如何解读

- **深红格子**：该代谢物与该临床指标正相关，代谢物浓度升高时该指标也升高，可能有生理功能关联
- **深蓝格子**：负相关关系，代谢物浓度升高时该指标降低
- **行模式**：某代谢物与多个表型指标均显著相关，提示该代谢物可能是重要的功能性生物标志物
- **列模式**：某临床指标与多个代谢物相关，说明该表型受多个代谢物共同影响
- **颜色块聚类**：颜色相似的区域可能代表一组代谢物共同关联某类临床指标

## 注意事项

- 相关性分析不等于因果关系，需结合生物学机制谨慎解读
- 多重检验校正（FDR）必须应用，否则大量的格子会出现假阳性显著性
- Spearman 相关适合非正态分布数据；正态分布数据可用 Pearson
- 样本量小（n<30）时相关系数不稳定，建议报告 95% 置信区间

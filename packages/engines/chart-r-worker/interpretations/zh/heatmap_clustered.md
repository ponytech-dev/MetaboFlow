# 样本×特征热图（Sample Heatmap）

## 图表说明

聚类热图以颜色编码展示差异代谢物在所有样本中的相对丰度，行（特征）和列（样本）均经层次聚类排序，旁边的树状图反映聚类结构。通常选取差异最显著的 Top N 个特征作图，颜色越深红表示丰度越高，蓝色表示丰度低于均值。

## 如何解读

- **行聚类（特征树）**：将表达模式相似的代谢物聚集在一起，揭示功能相关代谢物群
- **列聚类（样本树）**：将代谢谱相似的样本聚集，验证组内一致性和组间分离
- **色块模式**：若某一组样本在某类特征上呈现一致的红色或蓝色区块，表明该类代谢物具有稳定的组间差异
- **顶部/侧边注释条**：颜色条标注样本分组、批次等元数据，方便对应观察

## 注意事项

- 热图展示的是归一化后的相对丰度（Z-score），不代表绝对浓度
- Top N 阈值设置不同会影响聚类结果，建议同时查看火山图确认差异代谢物范围
- 若样本间组内变异很大（行内颜色杂乱），提示该组生物学异质性高或存在离群样本
- 特征名称若为质量-保留时间（m/z_RT），说明尚未完成注释

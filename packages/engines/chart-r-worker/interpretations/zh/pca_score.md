# PCA 得分图（PCA Score Plot）

## 图表说明

主成分分析（PCA）得分图是代谢组学数据质量评估和样本分群的首选无监督方法。将高维特征矩阵降维后，在 PC1 和 PC2 二维空间中展示每个样本的位置，同时标注各主成分解释的方差比例。椭圆为各组的 95% 置信区间。

## 如何解读

- **点的位置**：相似代谢谱的样本在图中距离接近；差异越大的样本距离越远
- **椭圆**：同一组样本的 95% 置信区间，椭圆重叠说明两组代谢谱差异不显著
- **PC1/PC2 解释方差**：比例越高说明该主成分捕获的信息越多；两轴合计 <50% 时需谨慎解读
- **QC 样本**：应聚集在图的中心区域，若分散则提示批次效应或系统漂移
- **异常点**：远离本组椭圆的样本可能是离群值，需检查原始数据

## 注意事项

- PCA 为无监督方法，无法保证组间分离；若分离不明显，不代表无差异代谢物
- 强批次效应会主导 PC1，掩盖真实生物学差异，建议先做批次校正再解读
- 默认使用 Pareto 或 UV 归一化，归一化方式会影响得分图形状
